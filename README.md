# -Olist-Brazilian-E-commerce-SQL-Analysis
## Project Overview
Analysis of the Olist Brazilian E-commerce dataset using MySQL.
Covers data loading, schema design, and business insights through SQL queries.

## Dataset
- Source: [Olist Dataset on Kaggle](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)
- Tables: customers, orders, order_items, products, payments, geolocation

## Files

 olist_schema.sql => DDL — table definitions and relationships 
 olist_etl.sql => Data loading and cleanup statements 
 olist_analysis_queries.sql=> Business analysis queries
 olist_sql_insights_report.pdf => 10 business questions with results and interpretation 

## Key Analysis
- Total revenue and monthly order trends
- Late delivery rate
- Top product categories and states by revenue
- Payment method breakdown
- Advanced queries using CTEs, window functions, and subqueries

## Tools Used
- MySQL
- MySQL Workbench
